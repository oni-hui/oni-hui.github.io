悬浮铁磁针动力学

《Dynamics of a Magnetic Needle Magnetometer》物理模型与关键公式推导笔记

说明：本笔记在之前公式推导的基础上，补充了详细的物理模型描述和直观图景，帮助你从“这到底是个什么装置、在干什么”角度理解论文。

0. 物理模型总览：这是什么装置？

0.1 磁针磁力计（MNM）的基本想法

论文研究的是一个单畴磁性针（magnetic needle, MN），把它当作一个“小小的刚体陀螺 + 巨大自旋磁矩”的系统，用来当磁力计（测磁场的仪器）。

  • 这根针是铁磁材料（比如钴），体积不大，但里面对齐的自旋很多，总自旋大小大约 \(S \sim 10^{12}\hbar\)（相当于 \(10^{12}\) 个电子自旋整齐指向同一方向）。
  • 这根针作为刚体，有一个易磁化轴（easy axis）：材料内部晶体各向异性使得自旋倾向沿某个固定方向排列，这个方向就是 \(n\)。
  • 针放在一个均匀的外磁场 \(B\) 中，自旋会绕着 \(B\) 进动（拉莫尔进动），这就是测量磁场的“信号”：进动频率 \(\omega_B = \gamma B\)。

核心问题：
这根针在真实环境中会有内部阻尼（Gilbert damping）和伴随的随机涨落（噪声 \(\xi\)）。这些噪声会让针的指向发生随机抖动，从而限制磁力计的分辨率（\(\Delta B\) 有下限）。论文的目标就是：

在给定材料参数（\(S,\omega_0,\alpha,T,\dots\)）下，计算仅由 Gilbert 阻尼及其噪声导致的磁场测量不确定度 \(\Delta B\)。

0.2 模型里的几个“角色”

  1. 刚体针的姿态：
    • \(n(t)\)：易轴方向的单位矢量，表示针在空间中朝向哪边。
    • 针作为刚体，有转动惯量张量 \(I = \text{diag}(I_X,I_Y,I_Z)\)（在体轴系中），这里假设是对称陀螺：\(I_X=I_Y\equiv I\), \(I_Z\neq I\)。
  2. 磁性：
    • 自旋算符 \(\hat S\)（总自旋角动量），对应的磁矩算符 \(\hat\mu = g\mu_B \hat S\)。
    • 有一个强的各向异性能量，使得 \(\hat S\) 喜欢沿着 \(\hat n\)（易轴）排列。
    • 大自旋 \(S\) 可以半经典地当作一个经典向量 \(S = S m\)，\(|m|\approx 1\)。
  3. 角动量与运动：
    • 轨道角动量（刚体旋转）：\(L\)。
    • 总角动量：\(J = L+S\)。
    • 由于磁矩在外磁场中受力矩，\(J\) 会随时间变化：
      • 自旋部分直接受 Zeeman 力矩；
      • 轨道部分通过各向异性耦合获得反作用力矩。
  4. 阻尼与噪声：
    • Gilbert 阻尼：自旋偏离易轴的分量会被内部自由度吸收（声子、磁振子等），表现为一个阻尼项，趋向于让 \(m\) 对齐 \(n\)。
    • 涨落-耗散定理：既然有阻尼（能量流向环境），就必然有相应的随机力矩 \(\xi(t)\) 把能量“噪声”地送回来。
    • \(\xi(t)\) 表现为一个附加“噪声磁场”，加在自旋方程上。

0.3 物理运动图景：陀螺 + 罗盘 + 带粘性 + 抖动

你可以把这个系统形象地看成：

  1. 一个陀螺，陀螺轴是 \(n\)，有转动惯量和角速度 \(\Omega\)。
  2. 陀螺内部有一根看不见的“磁针” \(m\)，通过强弹簧（各向异性）绑在陀螺轴上：\(m\) 偏离 \(n\) 会导致很大的恢复力矩（频率 \(\omega_0\)）。
  3. 外面有一个均匀磁场 \(B\)，对“磁针”施加 Zeeman 力矩，使 \(m\)（也几乎是 \(n\)）绕 \(B\) 进动。
  4. 系统内部有“粘性”（Gilbert 阻尼 \(\alpha\)），使 \(m\) 在很短时间内紧紧贴在 \(n\) 上；同时也有热噪声 \(\xi(t)\) 搅动，使 \(n\) 在长时间内缓慢随机扩散。

对磁力计来说：
我们观测的是 \(n\)（或 \(m\)）绕 \(b = B/|B|\) 的进动角度 \(\phi(t)\)。
理想情况下，\(\phi(t)\approx \omega_B t\)，我们凭这个反算出 \(B\)；
但现实中有随机扰动，\(\phi(t)\) 会出现扩散 \(\langle (\Delta\phi)^2\rangle\)，从而限制精度。

1. 量子模型与半经典极限 (1)–(5)

1.1 哈密顿量：刚体 + 各向异性 + Zeeman

针被视为一个对称刚体（symmetric top）加上一个大自旋。哈密顿量：

\[ H = \underbrace{\frac{1}{2I}\hat L^2 + \left(\frac{1}{2I_Z}-\frac{1}{2I}\right)\hat L_Z^2}_{H_R} \;-\;\underbrace{\frac{\omega_0}{\hbar}(\hat S\cdot\hat n)^2}_{H_A} \;-\;\underbrace{\hat\mu\cdot B}_{H_B}. \tag{1} \]

物理意义：

  • \(H_R\)：刚体的转动能量，\(\hat L\) 是绕质心的轨道角动量算符，\(\hat L_Z\) 是沿体轴 \(Z\) 的分量。当 \(I_X = I_Y = I \neq I_Z\)，就是对称陀螺。
  • \(H_A\)：各向异性能量，倾向于让 \(\hat S\) 对齐 \(\hat n\)（能量最低）；\(\omega_0\) 与材料参数 \(K\)（各向异性强度）、体积 \(V\) 等有关： \[ \omega_0 = 2\gamma^2 K S/V. \]
  • \(H_B\)：Zeeman 能量，\(\hat\mu = g\mu_B\hat S\)，所以 \(H_B = -g\mu_B\hat S\cdot B\)。

1.2 Heisenberg 方程与自旋/轨道方程

对算符 \(\hat A\)： \[ \frac{d\hat A}{dt} = \frac{i}{\hbar}[H,\hat A]. \]

利用角动量对易关系，可以推得：

  • 对自旋： \[ \boxed{ \dot{\hat S} = -g\mu_B B\times \hat S + 2\frac{\omega_0}{\hbar}(\hat S\times\hat n)(\hat S\cdot\hat n) } \tag{2} \] 物理解释：

    • 第一项：Zeeman 力矩，使自旋绕 \(B\) 进动。
    • 第二项：各向异性力矩，当自旋偏离易轴 \(n\) 时，有恢复力矩把它拉回去。

  • 对轨道角动量： \[ \boxed{ \dot{\hat L} = -2\frac{\omega_0}{\hbar}(\hat S\times\hat n)(\hat S\cdot\hat n) } \tag{3} \] 这恰好是自旋各向异性项的反向力矩，表示自旋偏离易轴时，会把角动量传给刚体转动（反作用）。

  • 对总角动量 \(\hat J = \hat L + \hat S\)： \[ \boxed{ \dot{\hat J} = -g\mu_B B\times \hat S } \tag{4} \] 只有外磁场提供的力矩改变总角动量，内部自旋-晶格耦合不改变总角动量。

  • 对易轴方向 \(\hat n\)： \[ \boxed{ \dot{\hat n} = I^{-1}\hbar^{-1}\left(\hat L\times \hat n + i\hbar\hat n\right) } \tag{5} \] 半经典极限下可以理解成 \(\dot n = \Omega\times n\)，其中 \(\Omega\) 是刚体角速度。

2. 半经典与无量纲化：从算符到 \((m,l,j,n)\) (6)–(10)

2.1 自旋很大：从量子到经典

因为 \(S\sim 10^{12}\hbar\) 非常大，系统很接近经典。取期望值： \[ S = \langle\hat S\rangle,\quad L = \langle\hat L\rangle,\quad n = \langle\hat n\rangle, \] 并假设涨落相对很小： \[ \sqrt{\langle\hat A^2\rangle - \langle\hat A\rangle^2} \ll |\langle\hat A\rangle|, \] 则可以把算符乘积视为平均值的乘积（mean-field approximation）。

2.2 无量纲向量与频率

定义无量纲向量： \[ m = \frac{S}{S},\quad l = \frac{L}{S},\quad j = \frac{J}{S} = m + l,\quad b = \frac{B}{|B|}. \]

定义频率： - 拉莫尔频率：\(\omega_B = \gamma |B|\)。 - 各向异性频率：\(\omega_0 = 2\gamma^2 K S/V\)。 - 刚体频率： \[ \omega_1 = \frac{S}{I_X},\quad \omega_3 = \frac{S}{I_Z}. \]

物理量级：论文中使用参数（类似钴）： - \(\omega_0 \sim 10^8\ \text{s}^{-1}\)（非常大） - \(\omega_1 \sim 10^2\ \text{s}^{-1}\)，\(\omega_3 \sim 7\times 10^3\ \text{s}^{-1}\)； - \(\alpha \sim 0.01\)，\(T\sim 300\ \text{K}\)，\(N = S/\hbar\sim 10^{12}\)。

2.3 半经典运动方程 (6)–(10)

经过无量纲化后，得到：

\[ \boxed{ \dot m = \omega_B m\times b + \omega_0 (m\times n)(m\cdot n) -\alpha\, m\times(\dot m - \Omega\times m) } \tag{6} \]

\[ \boxed{ \dot l = -\omega_0 (m\times n)(m\cdot n) + \alpha\, m\times(\dot m - \Omega\times m) } \tag{7} \]

\[ \boxed{ \dot n = \Omega\times n } \tag{8} \]

\[ \boxed{ \dot j = \omega_B m\times b } \tag{9} \]

\[ \boxed{ \Omega = (\omega_3-\omega_1)\big((j-m)\cdot n\big)n + \omega_1 (j-m). } \tag{10} \]

物理解读：

  • (6)：自旋单位矢量 \(m\) 的运动：

    • \(\omega_B m\times b\)：自旋绕磁场方向 \(b\) 的拉莫尔进动。
    • \(\omega_0(m\times n)(m\cdot n)\)：各向异性力矩，当 \(m\) 不沿 \(n\) 时，有强恢复力矩，频率 \(\omega_0\) 很大。
    • \(-\alpha m\times(\dot m - \Omega\times m)\)：Gilbert 阻尼，倾向于消除 \(m\) 在体轴系中的时间变化，使 \(m\) 慢慢贴着 \(n\) 旋转。

  • (7)：轨道角动量 \(l\) 的演化：

    • 与 (6) 中各向异性项和阻尼项正好相反，表示自旋-晶格耦合下的角动量交换。

  • (8)：刚体姿态 \(n\) 的演化，典型刚体公式 \(\dot n = \Omega\times n\)。

  • (9)：总角动量 \(j\) 仅由外 Zeeman 力矩改变。

  • (10)：角速度 \(\Omega\) 与 \(j-m\) 的关系：

    • \(j-m = l\) 是纯刚体角动量；
    • 与 \(n\) 的平行/垂直部分对应不同的惯量 \(I_Z,I_X\)，从而得到不同的转动频率 \(\omega_3,\omega_1\)。

重要物理结论：因为 \(\omega_0\gg\omega_{1,3,B}\) 且 \(\alpha\) 不小，\(m\) 很快被锁定在 \(n\) 上：
\[m(t)\approx n(t).\]

数值模拟（论文图 1–3）也验证了这一点：在短时间内 \(m\cdot n \to 1\)，之后 \(m\) 与 \(n\) 几乎重合。

3. 加入 Gilbert 噪声：随机 LLG 方程 (11)(12)

3.1 自旋阻尼与噪声：随机 Gilbert 模型

论文把 Brownian 运动的思路借到磁系统：
- 有粘性阻力（Gilbert 阻尼 \(\alpha\)）；
- 就必须有噪声力（内在磁场波动 \(\xi(t)\)）。

在 (6) 的磁场项中加入噪声： \[ \boxed{ \dot m = m\times(\omega_B b+\xi) + \omega_0 (m\times n)(m\cdot n) - \alpha\,m\times(\dot m - \Omega\times m). } \tag{11} \]

这里 \(\xi(t)\) 可以理解为一个随机的附加磁场（频谱与温度和阻尼相关），来自于： - 晶格振动（声子） - 磁振子（magnons） - 电流涨落 等内部自由度。

由于 \(\xi(t)\) 是内部噪声，对总角动量 \(j\) 的影响必须通过自旋-轨道耦合传递。因此在 \(l\) 的方程里加了 \(-m\times\xi\)，使得 (9) 仍无显式噪声：

\[ \dot j = \dot m + \dot l = \omega_B m\times b. \]

物理图景：
- 内部噪声直接扰动自旋（\(m\)），也间接扰动刚体（\(l\)）；
- 总角动量 \(j\) 只有通过 Zeeman 力矩间接感受到这些扰动。

3.2 噪声强度：涨落-耗散定理 (12)

涨落-耗散定理给出噪声 \(\xi\) 的频域相关函数： \[ \boxed{ \langle \xi_\alpha \xi_\beta\rangle_\omega \equiv \int dt\, e^{i\omega t}\langle\xi_\alpha(t)\xi_\beta(0)\rangle = \delta_{\alpha\beta}\,\frac{\alpha\omega}{N} \coth\left(\frac{\hbar\omega}{2k_B T}\right) \approx \delta_{\alpha\beta}\, \frac{2\alpha k_BT}{\hbar N} } \tag{12} \]

  • \(\alpha\) 越大，阻尼越强，噪声也越大。
  • \(N=S/\hbar\) 越大，总自旋越大，噪声越小（平均效果更好）。
  • 在室温下，\(\hbar\omega\ll k_B T\)，可用 \(\coth(x)\approx 1/x\) 近似，把频率依赖消除，噪声在低频近似为白噪声。

4. 消去快频率 \(\omega_0\)：慢变量的扰动展开 (13)–(17)

4.1 小参数 \(\lambda\) 与 adiabatic approximation

由于 \(\omega_0\) 最大，定义小参数： \[ \lambda \equiv \frac{\omega_1}{\omega_0}\ll 1. \]

直觉：

  • \(\omega_0\) 描述 \(m\) 被各向异性“弹簧”拉回 \(n\) 的频率，非常快。
  • \(\omega_1,\omega_3,\omega_B\) 描述刚体和进动的慢动力学。
  • 因此可将 \(m,n,j\) 等变量按 \(\lambda\) 展开，把超快过程消掉，只留慢变量的有效随机动力学。

展开： \[ m(t) = n_0(t) + \lambda\delta m(t) + \dots, \] \[ n(t) = n_0(t) + \lambda\delta n(t) + \dots, \] \[ j(t) = j_0(t) + \lambda\delta j(t) + \dots, \]

其中 \(n_0,j_0\) 是零阶解（一阶校正由 \(\delta m,\delta n,\delta j\) 给出）。因为阻尼足以让 \(m\) 快速对齐 \(n\)，零阶取： \[ m_0(t)\equiv n_0(t). \]

这意味着：在长时间尺度上（磁力计测量时间），可以认为 \(m\) 几乎总是贴着刚体轴 \(n\) 运动。

4.2 零阶动力学：对称陀螺方程 (13)

把展开代入 (8)(9)(10)，并用 adiabatic 条件消去 \(\Omega\) 的快变化，得到零阶方程：

\[ \boxed{ \dot j_0 = \omega_B n_0\times b,\qquad \dot n_0 = \omega_1 j_0\times n_0. } \tag{13} \]

物理解释：

  • \(\dot j_0 = \omega_B n_0\times b\)：
    • 磁矩基本沿 \(n_0\)，外磁场对 \(n_0\) 施加 Zeeman 力矩，使总角动量 \(j_0\) 在垂直于 \(n_0\) 的方向变化。
  • \(\dot n_0 = \omega_1 j_0\times n_0\)：
    • 针轴 \(n_0\) 在总角动量 \(j_0\) 周围以频率 \(\omega_1\) 进动（经典陀螺行为）。

论文指出 (13) 与对称陀螺在重力场中的运动方程等价（参见文中注 [12]），只不过这里“重力”被换成“磁场”，“重力中心距”被换成磁参数。

4.3 一阶扰动展开：导出 (14)(15)(16)(17)

目标：得到一阶方程，描述慢变量上因噪声 \(\xi\) 导致的随机扰动。这是用来计算扩散和 \(\Delta B\) 的关键。

用 \(\delta\eta\equiv\delta m-\delta n\) 表示“自旋方向 \(m\) 相对于针轴 \(n\) 的一阶偏离”，这是被各向异性弹簧锁住的快变量。最终我们会把 \(\delta\eta\) 消去，只留下 \(\delta n,\delta j\) 的有效方程。

4.3.1 从 (11) 到 (14)：\(\delta\eta\times n_0\) 的关系

将 \(m,n,j,\Omega\) 展开到 \(O(\lambda)\)，并利用零阶方程 (13) 把零阶项消去，论文得到：

\[ \omega_1(\delta m - \delta n)\times n_0 = \dot n_0 - \omega_B n_0\times b + \alpha n_0\times(\dot n_0 - \Omega_0\times n_0) - n_0\times\xi. \]

用 \(\dot n_0 = \omega_1 j_0\times n_0\) 和 \(\delta\eta\equiv\delta m-\delta n\)，得：

\[ \boxed{ \delta\eta\times n_0 = j_0\times n_0 - (\omega_B/\omega_1)\,n_0\times b - (1/\omega_1)\,n_0\times\xi. } \tag{14} \]

  1. 告诉你：
  • \(\delta\eta\)（\(m\) 与 \(n\) 的偏差）是如何被 \(j_0, b, \xi\) 共同驱动的。
  • 由于 \(\omega_0\) 很大，这个偏差最终会表现为一个小的、与噪声强度成比例的快速变量。

4.3.2 从 (9) 到 (15)：\(\delta j\) 的一阶方程

(9)：\(\dot j = \omega_B m\times b\)。展开到一阶：

\[ \dot j_0 + \lambda\dot{\delta j} = \omega_B (n_0+\lambda\delta m)\times b. \]

零阶匹配后，一阶部分为： \[ \dot{\delta j} = \omega_B\delta m\times b = \omega_B(\delta n + \delta\eta)\times b. \]

即：

\[ \boxed{ \frac{d}{dt}\delta j = \omega_B(\delta n + \delta\eta)\times b. } \tag{15} \]

含义：总角动量一阶扰动来自于针轴扰动 \(\delta n\) 与自旋-轴偏差 \(\delta\eta\)。

4.3.3 从 (8)(10) 到 (16)：\(\delta n\) 的一阶方程

(8)：\(\dot n = \Omega\times n\)，用 \(\Omega\) 的展开得到：

\[ \boxed{ \frac{d}{dt}\delta n = \omega_1 j_0\times\delta n + \omega_1(\delta j - \delta\eta)\times n_0. } \tag{16} \]

解释：

  • \(\omega_1 j_0\times\delta n\)：在零阶进动背景下，小扰动 \(\delta n\) 也绕 \(j_0\) 进动。
  • \(\omega_1(\delta j-\delta\eta)\times n_0\)：总角动量和自旋-轴偏差的扰动对针轴施加附加力矩。

4.3.4 正交条件与 (17)：整理 \(\delta j\) 的方程

由于 \(m,n\) 是单位矢量，一阶时有： \[ n_0\cdot\delta n = 0,\quad n_0\cdot\delta m = 0 \quad\Rightarrow\quad \delta\eta = \delta m-\delta n\perp n_0. \]

利用向量恒等式： \[ v\perp n_0\Rightarrow v = n_0\times(v\times n_0), \] 可以从 (14) 得到 \(\delta\eta\) 的显式表达，然后代入 (15)，论文最终整理为：

\[ \boxed{ \frac{d}{dt}\delta j = \omega_B\delta n\times b + \omega_B\big[j_0 - (j_0\cdot n_0)n_0\big]\times b - \frac{\omega_B^2}{\omega_1}(b\cdot n_0)n_0\times b + \frac{\omega_B}{\omega_1}\big[\xi - (\xi\cdot n_0)n_0\big]\times b. } \tag{17} \]

这条式子 + (13)(16)(14) 就构成了消去 \(\omega_0\) 后的慢变量随机动力学方程：
针轴 \(n_0\) 和总角动量 \(j_0\) 的运动中，包含了 Gilbert 噪声过滤后的有效扰动。

5. 旋转到绕 \(B\) 的坐标系：简化零阶解 (18)(19)

为了更容易处理 (13) 的零阶动力学，我们转到绕 \(b\) 以角速度 \(\omega_B\) 旋转的坐标系。

5.1 旋转系导数定义

对任意矢量 \(v\)，旋转系中的导数定义为： \[ \boxed{ \frac{d}{d\tau}v = \frac{dv}{dt} + \omega_B b\times v. } \]

这里 \(d/d\tau\) 表示在旋转参考系中的导数，\(\omega_B b\times v\) 是由参考系本身旋转引入的伪力项。

5.2 对 \(n_0,j_0\) 使用旋转导数

用 (13)： - 对 \(n_0\)： \[ \frac{d}{d\tau}n_0 = \dot n_0 + \omega_B b\times n_0 = \omega_1 j_0\times n_0 + \omega_B b\times n_0. \] 改写为 \(-\omega_1 n_0\times(\cdots)\) 形式： \[ \frac{d}{d\tau}n_0 = -\omega_1 n_0\times\left(n_0 - j_0 + \frac{\omega_B}{\omega_1}b\right). \] 即： \[ \boxed{ \frac{d}{d\tau} n_0 = -\omega_1 n_0\times\left(n_0 - j_0 + \frac{\omega_B}{\omega_1} b\right). } \tag{18} \]

  • 对 \(j_0\)： \[ \frac{d}{d\tau}j_0 = \dot j_0 + \omega_B b\times j_0 = \omega_B n_0\times b + \omega_B b\times j_0 = \omega_B b\times\left(n_0 - j_0 + \frac{\omega_B}{\omega_1} b\right), \] 即： \[ \boxed{ \frac{d}{d\tau} j_0 = \omega_B b\times\left(n_0 - j_0 + \frac{\omega_B}{\omega_1} b\right). } \tag{19} \]

5.3 特殊初始条件与零阶常量解

如果初始时刻满足： \[ n_0(0) - j_0(0) + \frac{\omega_B}{\omega_1}b = 0, \] 那么在旋转系中 (18)(19) 中的括号恒为 0，因此： \[ \frac{d}{d\tau}n_0 = 0,\quad \frac{d}{d\tau}j_0 = 0, \] 即 \(n_0,j_0\) 在旋转系中是常向量。这样一来，一阶扰动方程中的系数矩阵就是常数矩阵，极大简化了分析。

6. 常系数随机线性系统与矩阵指数解 (20)–(22)

在上述特殊初始条件下，旋转系中 \(n_0,j_0,b\) 是常量。一阶扰动方程 (20)(21) 为：

\[ \frac{d}{d\tau}\delta n = \omega_1 n_0\times(\delta n - \delta j + \delta\eta), \tag{20} \] \[ \frac{d}{d\tau}\delta j = -\omega_B b\times(\delta n - \delta j + \delta\eta). \tag{21} \]

配合简化后的 (14)： \[ \delta\eta\times n_0 = -\frac{1}{\omega_1}n_0\times\xi, \] 可以把 \(\delta\eta\) 视为一个线性噪声源。

6.1 写成矩阵形式

定义 6 维向量： \[ X(\tau) \equiv \begin{pmatrix} \delta n(\tau)\\[3pt]\delta j(\tau) \end{pmatrix}, \] 定义 3×3 矩阵： \[ A_{ij} = -\omega_1\epsilon_{ijk}n_{0k},\quad B_{ij} = -\omega_B\epsilon_{ijk}b_k. \]

则 (20)(21) 可写成： \[ \frac{d}{d\tau} \begin{pmatrix} \delta n\\ \delta j \end{pmatrix} = \begin{pmatrix} -A & A\\ B & -B \end{pmatrix} \begin{pmatrix} \delta n\\ \delta j \end{pmatrix} + \begin{pmatrix} -A\\ B \end{pmatrix}\delta\eta. \]

论文定义一个 6×6 常矩阵： \[ \boxed{ C = \begin{pmatrix} A & -B\\ -A & B \end{pmatrix}, } \]

并把噪声源写成： \[ f(t) = \begin{pmatrix} \delta\eta(t)\\[3pt]0 \end{pmatrix}. \]

于是线性系统一般形式为： \[ \frac{d}{d\tau}X(\tau) = C X(\tau) + f(\tau). \]

6.2 矩阵指数解：式 (22)

线性常系数微分方程： \[ \frac{d}{dt}X(t) = C X(t) + f(t),\quad X(0) = X_0. \]

通解： \[ X(t) = e^{Ct}X_0 + \int_0^t ds\, e^{C(t-s)} f(s). \]

这里初始条件为 \(\delta n(0)=0,\delta j(0)=0\)，即 \(X(0)=0\)，故：

\[ \boxed{ \begin{pmatrix} \delta n(t)\\[3pt] \delta j(t) \end{pmatrix} = \int_0^t dt_1\, \exp\big[C(t-t_1)\big] \begin{pmatrix} \delta\eta(t_1)\\[3pt]0 \end{pmatrix}. } \tag{22} \]

这是论文用来计算针轴扰动的核心公式。

7. 角度扩散与磁场灵敏度 (23)：物理含义

7.1 关心的量：\(\langle\delta n_y^2(t)\rangle\) 与进动角度

选坐标系： - \(n_0\) 沿 \(\hat z\)：\(n_0 = \hat z\)； - \(b\) 在 \(xz\) 平面，与 \(\hat z\) 夹角 \(\theta\)： \[ b = \cos\theta\,\hat z + \sin\theta\,\hat x. \]

在这个基底下，\(y\) 方向就是 \(n_0\) 绕 \(b\) 进动的主要摆动方向。因此：

  • \(\delta n_y(t)\) 对应针轴在垂直于进动的方向上的随机偏移；
  • 对小角度近似下，\(\delta\phi \sim \delta n_y\)，于是 \[ \langle(\Delta\phi)^2\rangle \approx \langle\delta n_y^2(t)\rangle. \]

利用 (22) 和噪声相关（经过一系列频域计算，论文在检索片段中有详细说明），得到：

\[ \langle\delta n_y^2(t)\rangle \approx t\,\omega_1^2 S_a(\omega\sim\omega_1), \] 其中 \(S_a(\omega)\) 是 \(\delta\eta\) 的有效谱密度，满足（由 (12) 与几何关系推得）：

\[ S_a(\omega) \approx \frac{\alpha\omega}{\omega_0^2 N} \coth\left(\frac{\hbar\omega}{2k_B T}\right) \approx \frac{2\alpha k_B T}{\hbar N\omega_0^2}. \]

于是角度扩散： \[ \langle(\Delta\phi)^2\rangle \approx t\,\omega_1^2 \frac{2\alpha k_BT}{\hbar N\omega_0^2}. \]

7.2 频率与磁场的不确定度

我们用时间 \(t\) 来估计拉莫尔频率 \(\omega_B\)，角度上的不确定度转化为频率不确定度： \[ \Delta\omega_B \approx \frac{\sqrt{\langle(\Delta\phi)^2\rangle}}{t} = \left(\frac{\omega_1}{\omega_0}\right) \sqrt{\frac{2\alpha k_BT}{\hbar N}}\cdot \frac{1}{\sqrt{t}}. \]

磁场与频率关系：\(\omega_B = \gamma B\)，因此： \[ \Delta B = \frac{\Delta\omega_B}{\gamma}. \]

因为 \(\gamma = g\mu_B/\hbar\)，得 \(1/\gamma = \hbar/(g\mu_B)\)，代入得到论文的中心结果：

\[ \boxed{ \Delta B = \frac{\Delta\omega_B}{\gamma} \approx \frac{\hbar}{g\mu_B} \cdot\frac{\omega_1}{\omega_0} \sqrt{\frac{2\alpha k_BT}{\hbar N}}\cdot\frac{1}{\sqrt{t}}. } \tag{23} \]

物理解读：

  • \(\Delta B \propto t^{-1/2}\)：典型的扩散极限（随测量时间开方改善）。
  • \(\Delta B\) 随：
    • \(\alpha\) 增大而变差（阻尼越大，噪声越大）；
    • \(T\) 增大而变差（温度越高，热噪声越大）；
    • \(N\) 增大而变好（更多自旋平均噪声）；
    • \(\omega_0\) 增大而变好（易轴越硬，自旋偏离越小）；
    • \(\omega_1/\omega_0\) 比率越小越好（陀螺进动越慢、各向异性越快）。

8. 总结：把数学与物理图景对上号

  1. 物理模型：
    • 单畴磁针 = 刚体陀螺 + 巨大自旋 + 强易轴各向异性；
    • 外磁场 \(B\) 使针轴绕 \(B\) 进动；
    • 内部 Gilbert 阻尼 + 相应噪声 \(\xi\) 导致针轴方向 \(n\) 的随机扩散。
  2. 数学结构：
    • 由量子哈密顿量 (1) 得到 Heisenberg 方程 (2)–(5)；
    • 取半经典平均并无量纲化，得到 LLG 型方程 (6)–(10)；
    • 加入噪声，得到随机 LLG 方程 (11) 和噪声谱 (12)；
    • 利用 \(\omega_0\gg\omega_{1,3,B}\) 做 adiabatic 展开：
      • 零阶：\(m\approx n_0\)，满足陀螺方程 (13)；
      • 一阶：得到 \(\delta\eta,\delta n,\delta j\) 的随机线性方程 (14)(15)(16)(17)；
    • 在绕 \(B\) 的旋转系中，零阶解变为常量 (18)(19)；
    • 一阶扰动形成常系数随机线性系统，矩阵指数解为 (22)。
  3. 物理输出：
    • 通过 (22)+噪声谱计算 \(\langle\delta n_y^2(t)\rangle\)；
    • 进动角度扩散 \(\langle(\Delta\phi)^2\rangle\approx t\omega_1^2 S_a\)；
    • 频率/磁场不确定度 \(\Delta\omega_B,\Delta B\) 给出 Gilbert 噪声的极限敏感度 (23)。

从磁力计应用角度看，这篇论文的结论非常乐观：
在内部 Gilbert 噪声的限制下，磁针磁力计的理论极限灵敏度极高，实际实验中更可能是外部噪声（如环境磁场波动、器件噪声）占主导。
