铁磁扭矩传感器笔记

基于实验现象的动力学模型建模

前文考虑I型超导体微倾斜悬浮动力学模型得到的模拟结果仍然与实际实验现象相去甚远。

分析了一个更常见且易于实现的场景：I型超导体产生的悬浮势阱对称性破缺情况下，悬浮铁磁体呈倾斜状态，悬浮磁体做纯摆动运动的动力学模型

考虑一个悬浮的硬铁磁体；为简单起见，我们采用如参考文献 [8, 9, 10] 中的球体。设 \(\mu=-\gamma_{0}S\) 为磁偶极矩，其中 \(S\) 是总自旋角动量，\(\gamma_{0}\) 是旋磁比。我们假设，由于悬浮机制的性质，磁矩被束缚在 \(x\) 方向 [图 1(a)]，对于绕 \(z\) (\(\alpha\) 角) 和 \(y\) (\(\beta\) 角) 的小幅度振动，具有简谐恢复扭矩，由 \(\tau_{z}=-I\omega_{\alpha}^{2}\alpha\) 和 \(\tau_{y}=I\omega_{\beta}^{2}\beta\) 给出。这里，\(I\) 是各向同性转动惯量，\(\omega_{\alpha}\)、\(\omega_{\beta}\) 是振动模式的共振频率。

对于小角度 \(\alpha\)、\(\beta\)，在线性近似中，我们可以将总自旋写为 \(S\simeq S(1,\alpha,\beta)\)，角速度矢量写为 \(\Omega\simeq(\dot{\gamma},-\dot{\beta},\dot{\alpha})\)，其中我们引入了对应于绕 \(S\) 旋转的角度 \(\gamma\)。最后，我们假设磁体质心处于稳定平衡位置，并忽略平移和振动自由度之间的交叉耦合。

该模型描述的情况近似于最近的实验实现，其中磁性微球体悬浮在 II 型 [8, 9] 或 I 型 [10] 超导体上方。在前一种情况下，平移和振动束缚是由钉扎到涡旋产生的。在后一种情况下，它们由陷阱的几何形状决定。例如，对于在没有涡旋的情况下放置在超导平面上方的磁偶极子，发现水平位置 \(\beta=0\) 是稳定的 [图 1(b)]。恢复扭矩由迈斯纳场提供，这等效于由镜像偶极子产生的场。解析计算 [10] 表明迈斯纳场由下式给出：

\[B_{i}=-\frac{\mu_{0}}{32\pi z_{0}^{3}}(\mu_{x},\mu_{y},2\mu_{z}),\]

其中 \(z_{0}\) 是超导体上方的平衡高度。对于小 \(\beta\)，场的大小为 \(B_{i}\simeq \mu_{0}\mu/(32\pi z_{0}^{3})\)，恢复频率可以写为：

\[\omega_{\beta}^{2}=\omega_{L}\omega_{I},\]

其中 \(\omega_{L}=\gamma_{0}B_{i}\) 是在场 \(B_{i}\) 中的拉莫尔频率，\(\omega_{I}=S/I\) 是与自旋角动量 \(S\) 相关的爱因斯坦-德哈斯频率。对于 \(\alpha\) 自由度，束缚可能源于不同的原因。例如，在参考文献 [10] 中，尽管陷阱具有圆柱对称性，但由于倾斜效应引起的微小对称性破缺，发现 \(\alpha\) 运动是被束缚的。

遵循参考文献 [5]，我们将偶极子的角运动方程写为：

\[\dot{\boldsymbol{J}}=\boldsymbol{\tau}_{i}+\boldsymbol{\tau},\]

\[\dot{\boldsymbol{S}}=\Omega\times S,\]

其中 \(\boldsymbol{J}=\boldsymbol{L}+\boldsymbol{S}\) 是总角动量，\(\boldsymbol{L}=I\Omega\) 是动力学角动量；\(\boldsymbol{\tau}_{i}=\mu\times B_{i}\) 是迈斯纳场 \(B_{i}\) 的恢复扭矩；\(\boldsymbol{\tau}\) 是外部扭矩。后者可能由额外的场 \(\boldsymbol{B}\) 引起。方程 (6) 严格适用于硬铁磁体，即假设自旋始终锁定在各向异性轴上，因此刚性地附着在磁体晶体上 [11, 5]。

在线性化极限下，运动方程可以用角度 \(\alpha\) 和 \(\beta\) 表示为：

\[\ddot{\alpha}=-\omega_{\alpha}^{2}\alpha-\frac{\omega_{\alpha}}{Q_{\alpha}}\dot{\alpha}-(\omega_{I}+\dot{\gamma})\dot{\beta}+\frac{\tau_{z}}{I},\]

\[\ddot{\beta} = -\omega_{\beta}^2 \beta - \frac{\omega_{\beta}}{Q_{\beta}} \dot{\beta} + (\omega_I + \dot{\gamma}) \dot{\alpha} - \frac{\tau_y}{I},\]

这是一个两个动力学耦合的谐振子系统。这里，通过定义唯象品质因数 \(Q_a\) 和 \(Q_\beta\) 手动添加了阻尼。交叉耦合的动力学项是陀螺行为的特征。它们包含一个与 \(\dot{\gamma}\) 成正比的经典陀螺项和一个与 \(\omega_I\) 成正比的爱因斯坦-德哈斯项，后者表达了量子自旋是真正的角动量这一事实 [12]。实际上，在没有恢复力的情况下，系统预计会表现为一个陀螺仪，展示出进动和章动 [1,5]。

从现在开始，我们假设系统参数使得交叉耦合项可以忽略。假设固有旋转 \(\dot{\gamma}\) 可忽略，一个充分条件是 \(\omega_I \ll \omega_{a,\beta}\)。如果与 \(\omega_I\) 成正比的陀螺项显著但相对较小，方程 (7) 和 (8) 的线性系统仍然可以通过适当的坐标变化进行对角化。另一方面，当 \(\omega_I \gg \omega_{a,\beta}\) 时，陀螺项变得优于恢复项，线性化近似失效，需要像其他地方所做的那样分析完整的非线性陀螺动力学 [1]。在参考文献 [6] 中分析了一种中间情况，其中 \(\beta\) 角是振动束缚的，而 \(\alpha\) 角是自由的，导致了一种有趣的混合振动-进动状态。

接下来，我们评估悬浮磁体对外部扭矩 \(\tau_z\) 的灵敏度，特别是在这由外部磁场 \(B\) 引起的情况下。对于可忽略的陀螺效应，方程是简单的解耦谐振子。不失一般性，我们关注 \(\alpha\) 自由度，并假设一个最优定向的磁场 \(B = B(0, 1, 0)\) 沿 y 轴施加，如图 1(a) 所示，导致沿 z 方向的扭矩 \(\tau = \mu \times B\)。转到傅里叶空间，运动方程 (7) 变为：

\[\alpha(\omega) = \chi(\omega) \tau_z(\omega),\]

其中我们定义了振动敏感性：

\[\chi(\omega) = \frac{1}{I(-\omega^2 + \omega_a^2 + \frac{i\omega_a \omega_a}{Q_a})},\]

并且

\[\tau_z(\omega) = -\mu B(\omega)\]

是扭矩的 z 分量。通过监测振动运动，我们可以推断出扭矩，并通过方程 (11) 推断出施加的场 \(B\)。因此，我们可以通过纯粹的机械测量来测量 \(B\)。

纯摆动模式下灵敏度分析

机械测量中唯一的基本噪声源是热机械噪声和量子探测噪声 [7]。根据涨落-耗散定理 [13]，热噪声可以通过单边谱密度建模：

\[S_{\tau_y} = 4k_B T \text{Im}[\chi(\omega)^{-1}]/\omega = 4k_B T I \frac{\omega_a}{Q_a},\]

其中 \(k_B\) 是玻尔兹曼常数，\(T\) 是环境温度，Im 表示虚部。由 \(\chi(\omega)\) 的虚部表示的耗散包括所有机械耗散源，例如，由悬浮磁体中的畴磁损耗和磁振子引起的耗散，这可能为这类传感器设定了最终极限 [10]。

对于扭矩量子噪声，我们采用通常应用于线性相位不敏感力检测的方法 [14]（详细推导见补充材料 [15]），得到扭矩的标准量子极限：

\[S_{\tau_s \omega} \geq 2\hbar|\chi(\omega)|^{-1} = 2\hbar I \left[(-\omega^2 + \omega_a^2)^2 + \left(\frac{\omega_a \omega_a}{Q_a}\right)^2\right]^{1/2}.\]

SQL 通常对任何海森堡极限的线性运动探测器都有效，无论其实现方式如何。可能的探测方案可以是基于 SQUID 的 [10]、光学的 [8,9,16] 或微波的 [17]。在所有情况下，SQL 在原则上都是可以实现的。对于 SQUID，已经证明了噪声因子小于 10 倍 SQL 的情况 [18,19]。值得注意的是，与 ERL 类似，机械测量的 SQL 也不是一个基本极限，并且在原则上可以被超越 [20]。然而，在这里它仍然可以被视为一个有用的基准，因为我们有兴趣评估机械探测的 SQL 是否意味着铁磁扭矩磁力计中的 ERL。如下所示，我们发现 SQL 可以远低于 ERL。

从方程 (11) 中，我们可以将热噪声和 SQL 扭矩噪声转换为等效的磁场噪声。就功率谱密度而言，关系很简单 \(S_B = S_\tau / \mu^2\)，这里我们仍然假设场是最优定向的。在图 2(a) 中，我们绘制了 SQL、热噪声和 ERL 噪声 [方程 (2)]，所有噪声都表示为磁场幅度谱密度 \(\sqrt{S_B}\)，参数采用参考文献 [10] 中展示的现实参数，即一个铁磁 NdFeB 球体，\(R = 30 \, \mu m\)，通过迈斯纳效应在 \(T = 4.2 \, K\) 下悬浮，振动品质因数 \(Q = 10^7\)。对于此设置，我们估计爱因斯坦-德哈斯频率 \(f_I = \omega_I / 2\pi = 0.188 \, Hz\) [图 2(a) 中的垂直线]。振动频率设置为 \(f_a = \omega_a / 2\pi = 10 f_I = 1.88 \, Hz\)。这个值比当前实验低一个数量级，预计可以通过适当控制
